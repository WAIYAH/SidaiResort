## How to Update the Site
1. Edit the file you want to change
2. Save the file
3. Open terminal: git add . && git commit -m "what you changed" && git push
4. Cloudflare auto-deploys in ~30 seconds — done

## Adding New Images
Put images in assets/images/ in the correct subfolder:
branding/ | hero/ | rooms/ | dining/ | experiences/ | events/ | nature/ | facility/
